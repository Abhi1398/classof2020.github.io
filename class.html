Add Member to the List

This website is open source, therefore anyone in the class can submit edits through pull requests. If you or your friend in the class isn’t on this list, but should be, please add it:

  1. Browse to the website repository: here
  2. Fork and clone the repository(or clone it if you have commit access)
  3. Create a branch.
  4. Navigate to the _class/ directory. Open _TEMPLATE.md and follow the instructions there.
  5. After you've made the necessary changes, commit the feature branch.
  6. Push the feature branch to GitHub.
  7. Open a pull request to merge the branch.
  8. Click “Create pull request”
